Abstract

This report analyses five independently collected datasets related to human well-being, stress, and productivity. Pearson correlation identifies variables with the strongest direct linear association with each target, while Principal Component Analysis (PCA) reveals the dominant structure of each dataset. Results consistently highlight sleep quality, stress level, physical activity, and work-life balance as key recurring factors across all five contexts. The analysis is purely exploratory: findings describe statistical associations, not causal relationships.

1 Introduction

The goal of this project is to identify which lifestyle, sleep, and work-related variables are most strongly associated with well-being, stress, and productivity. The analysis is based on five separate datasets, each representing a different population and context (Kumar 2025; Tharmalingam 2023; Mnaadi 2023; Rogg 2023; A. 2024). Because the datasets come from different sources and cannot be meaningfully merged, each one is analysed independently; the results are then compared in Section 5 to surface recurring themes.

The two analytical methods applied throughout are Pearson correlation (described in Section 3.1) and Principal Component Analysis (described in Section 3.3). Both methods are implemented using the scikit-learn library (Pedregosa et al. 2011) and data manipulation relies on pandas (McKinney 2010).

2 Datasets

The five cleaned datasets used in this project are listed in Table 1.

Each dataset has its own target variable; this variable is excluded from the predictor set during analysis and treated as the dependent measure. The datasets vary considerably in size (100 to 20 000 observations) and in the specific variables they contain, which is why a unified model is not appropriate.

3 Methods

3.1 Correlation Analysis

For each dataset, the Pearson correlation coefficient is computed between every numeric predictor and the selected target variable. The Pearson coefficient measures the strength and direction of a linear relationship between two continuous variables and is defined as:

\[ r_{xy} = \frac{\displaystyle\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})} {\sqrt{\displaystyle\sum_{i=1}^{n}(x_i - \bar{x})^2} \cdot \sqrt{\displaystyle\sum_{i=1}^{n}(y_i - \bar{y})^2}} \tag{1}\]

where \(x_i\) and \(y_i\) are paired observations, and \(\bar{x}\), \(\bar{y}\) are their respective sample means. A value of \(r = +1\) indicates a perfect positive linear relationship; \(r = -1\) a perfect negative one; \(r = 0\) no linear association. Results should not be interpreted as evidence of causation.

3.2 Standardisation

Before PCA, all predictor columns are standardised to zero mean and unit variance using the z-score transformation defined in Equation 2. This step is essential because PCA is sensitive to the scale of the input variables — without standardisation, variables with large numeric ranges would dominate the principal components regardless of their statistical relevance.

\[ z_i = \frac{x_i - \mu}{\sigma} \tag{2}\]

where \(\mu\) is the sample mean and \(\sigma\) is the sample standard deviation of the variable.

3.3 Principal Component Analysis

PCA is an unsupervised dimensionality-reduction method that transforms the original \(p\) numeric variables into a new orthogonal coordinate system (Jolliffe and Cadima 2016). The first principal component (PC1) captures the largest possible fraction of the total variance; PC2 captures the most remaining variance while being orthogonal to PC1, and so on.

The fraction of total variance explained by component \(k\) is called the explained variance ratio (EVR):

\[ \text{EVR}_k = \frac{\lambda_k}{\displaystyle\sum_{j=1}^{p}\lambda_j} \tag{3}\]

where \(\lambda_k\) is the \(k\)-th eigenvalue of the covariance matrix of the standardised data (see Equation 2 for standardisation).

Three PCA outputs are reported for each dataset: (1) the explained variance plot (Equation 3), (2) the PC1 loadings, and (3) a scatter of observations in the PC1–PC2 plane.

5 General Comparison

Table 7 below summarises the strongest positive and negative correlation with the target variable in each dataset. This cross-dataset view makes it possible to identify variables that recur as important predictors across multiple contexts.

Code 
_datasets_info = [
    ("Lifestyle",   df_pca,      "wellness"),
    ("Productivity",df_prod,     "productivity"),
    ("Sleep",       df_sleep,    "stress_level"),
    ("Students",    df_students, "stress_numeric"),
    ("Remote Work", df_remote,   "productivity_score"),
]

rows = []
for name, df, tgt in _datasets_info:
    corr = target_correlations(df, tgt)
    pos = corr[corr > 0]
    neg = corr[corr < 0]
    top_pos = f"{pos.idxmax()} ({pos.max():.2f})" if len(pos) > 0 else "—"
    top_neg = f"{neg.idxmin()} ({neg.min():.2f})" if len(neg) > 0 else "—"
    rows.append({
        "Dataset": name,
        "Target": tgt,
        "Strongest positive": top_pos,
        "Strongest negative": top_neg,
    })

display(Markdown(pd.DataFrame(rows).to_markdown(index=False)))

As Table 7 demonstrates, several variable types recur across datasets:

  • Sleep-related variables appear in the lifestyle, sleep, student, and remote work datasets, consistently showing a positive relationship with well-being indicators.
  • Stress level appears in multiple datasets as the strongest negative predictor, which is expected given that all targets are measures of well-being or performance.
  • Physical activity shows positive associations in the lifestyle and student datasets.
  • Work-life balance ranks highly in the remote work dataset.

This cross-dataset consistency provides stronger evidence that these factors are genuinely associated with well-being, rather than being artefacts of any single sample.

6 Limitations

Several limitations should be acknowledged:

  1. Different populations and measurement scales. The five datasets describe different groups of people using different instruments. Direct numeric comparison of correlation coefficients across datasets may be misleading.
  2. Correlation ≠ causation. Equation 1 measures linear association only. No causal inference is warranted from these results.
  3. Sample sizes vary greatly. The remote work dataset has only 100 observations, making its PCA results (see Section 4.5) less reliable than those for the productivity dataset (n = 20 000).
  4. Missing variables. Stress, sleep, and activity are consistently important, but many potential confounders (age structure, socioeconomic status, cultural context) are absent from most datasets.

7 Conclusion

This analysis examined five datasets related to human well-being, stress, and productivity using Pearson correlation (Equation 1) and PCA (Equation 3). The standardisation step (Equation 2) ensured that PCA results are scale-invariant.

Across all five datasets (summarised in Table 7 and Table 1), a consistent picture emerges: sleep quality and duration, stress level, physical activity, and work-life balance are the variables most frequently and most strongly associated with the respective target variables. These factors appear in contexts as different as student life, clinical sleep studies, and remote work, which makes the pattern more robust.

The analysis has been conducted in a purely exploratory manner. })

_ID_COLS = {"person_id", "employee_id"}

def numeric_features(df, target):
    num = df.select_dtypes(include="number").columns.tolist()
    return [c for c in num if c != target and c not in _ID_COLS]

def target_correlations(df, target):
    feats = numeric_features(df, target)
    return df[feats + [target]].corr(numeric_only=True)[target].drop(target)

def run_pca(df, target):
    feats = numeric_features(df, target)
    X = df[feats].dropna()
    scaler = StandardScaler()
    X_scaled = scaler.fit_transform(X)
    pca = PCA()
    scores = pca.fit_transform(X_scaled)
    loadings = pd.DataFrame(
        pca.components_.T,
        index=feats,
        columns=[f"PC{i}" for i in range(1, len(feats) + 1)],
    )
    return scores, loadings, pca.explained_variance_ratio_

def plot_correlations(df, target, title):
    corr = target_correlations(df, target).sort_values()
    colors = ["#B65E5A" if v < 0 else "#4C78A8" for v in corr.values]
    h = max(3.5, len(corr) * 0.45)
    fig, ax = plt.subplots(figsize=(9, h))
    ax.barh(corr.index, corr.values, color=colors, edgecolor="none")
    ax.axvline(0, color="#444", linewidth=1)
    ax.set_title(title)
    ax.set_xlabel("Pearson correlation")
    for i, v in enumerate(corr.values):
        ha = "left" if v >= 0 else "right"
        offset = 0.012 if v >= 0 else -0.012
        ax.text(v + offset, i, f"{v:.2f}", va="center", ha=ha, fontsize=9)
    plt.tight_layout()
    plt.show()

def plot_variance(var_ratio, title):
    pcs = np.arange(1, len(var_ratio) + 1)
    cum = np.cumsum(var_ratio)
    fig, ax = plt.subplots(figsize=(9, 4.5))
    bars = ax.bar(pcs, var_ratio, color="#4C78A8", width=0.7, edgecolor="none",
                  label="Per-component")
    ax.plot(pcs, cum, marker="o", color="#222", linewidth=2, label="Cumulative")
    ax.set_xticks(pcs)
    ax.set_title(title)
    ax.set_xlabel("Principal component")
    ax.set_ylabel("Variance ratio")
    for rect, v in zip(bars, var_ratio):
        ax.text(rect.get_x() + rect.get_width() / 2,
                rect.get_height() + 0.01, f"{v:.2f}", ha="center", fontsize=8)
    ax.legend()
    plt.tight_layout()
    plt.show()

def plot_loadings(loadings, title):
    pc1 = loadings["PC1"].sort_values()
    colors = ["#B65E5A" if v < 0 else "#59A14F" for v in pc1.values]
    h = max(3.5, len(pc1) * 0.45)
    fig, ax = plt.subplots(figsize=(9, h))
    ax.barh(pc1.index, pc1.values, color=colors, edgecolor="none")
    ax.axvline(0, color="#444", linewidth=1)
    ax.set_title(title)
    ax.set_xlabel("Loading")
    for i, v in enumerate(pc1.values):
        ha = "left" if v >= 0 else "right"
        offset = 0.012 if v >= 0 else -0.012
        ax.text(v + offset, i, f"{v:.2f}", va="center", ha=ha, fontsize=9)
    plt.tight_layout()
    plt.show()

def plot_pc_scatter(scores, title, max_pts=3000):
    rng = np.random.RandomState(42)
    s = scores
    if len(s) > max_pts:
        idx = rng.choice(len(s), max_pts, replace=False)
        s = s[idx]
    fig, ax = plt.subplots(figsize=(9, 5.5))
    ax.scatter(s[:, 0], s[:, 1], s=25, alpha=0.55,
               color="#4C78A8", edgecolors="none")
    ax.axhline(0, color="#BDBDBD", linewidth=1)
    ax.axvline(0, color="#BDBDBD", linewidth=1)
    ax.set_title(title)
    ax.set_xlabel("PC1")
    ax.set_ylabel("PC2")
    plt.tight_layout()
    plt.show()

def corr_table(df, target):
    corr = target_correlations(df, target).sort_values(ascending=False)
    out = corr.reset_index()
    out.columns = ["Feature", "Pearson r"]
    out["Pearson r"] = out["Pearson r"].round(3)
    display(Markdown(out.to_markdown(index=False)))

DATA_DIR = "data/clean_data"

## Correlation Analysis {#sec-methods-corr}

For each dataset, the Pearson correlation coefficient is computed between every
numeric predictor and the selected target variable. The Pearson coefficient

$$

               \sqrt{\displaystyle\sum_{i=1}^{n}(y_i - \bar{y})^2}}
$$ {#eq-pearson}

where $x_i$ and $y_i$ are paired observations, and $\bar{x}$, $\bar{y}$ are

$$

$$ {#eq-zscore}

where $\mu$ is the sample mean and $\sigma$ is the sample standard deviation

## Principal Component Analysis {#sec-methods-pca}

PCA is an unsupervised dimensionality-reduction method that transforms the

$$

where $\lambda_k$ is the $k$-th eigenvalue of the covariance matrix of the

Three PCA outputs are reported for each dataset: (1) the explained variance

# General Comparison {#sec-comparison}

@tbl-summary below summarises the strongest positive and negative correlation
with the target variable in each dataset. This cross-dataset view makes it
possible to identify variables that recur as important predictors across
multiple contexts.

```{python}
#| label: tbl-summary
#| tbl-cap: "Strongest positive and negative Pearson correlation with the target variable in each dataset."
_datasets_info = [
    ("Lifestyle",   df_pca,      "wellness"),
    ("Productivity",df_prod,     "productivity"),
    ("Sleep",       df_sleep,    "stress_level"),
    ("Students",    df_students, "stress_numeric"),
    ("Remote Work", df_remote,   "productivity_score"),
]

rows = []
for name, df, tgt in _datasets_info:
    corr = target_correlations(df, tgt)
    pos = corr[corr > 0]
    neg = corr[corr < 0]
    top_pos = f"{pos.idxmax()} ({pos.max():.2f})" if len(pos) > 0 else "—"
    top_neg = f"{neg.idxmin()} ({neg.min():.2f})" if len(neg) > 0 else "—"
    rows.append({
        "Dataset": name,
        "Target": tgt,
        "Strongest positive": top_pos,
        "Strongest negative": top_neg,
    })

display(Markdown(pd.DataFrame(rows).to_markdown(index=False)))
```

As @tbl-summary demonstrates, several variable types recur across datasets:

- **Sleep-related variables** appear in the lifestyle, sleep, student, and
  remote work datasets, consistently showing a positive relationship with
  well-being indicators.
- **Stress level** appears in multiple datasets as the strongest negative
  predictor, which is expected given that all targets are measures of
  well-being or performance.
- **Physical activity** shows positive associations in the lifestyle and
  student datasets.
- **Work-life balance** ranks highly in the remote work dataset.

This cross-dataset consistency provides stronger evidence that these factors
are genuinely associated with well-being, rather than being artefacts of any
single sample.

# Limitations

Several limitations should be acknowledged:

1. **Different populations and measurement scales.** The five datasets describe
   different groups of people using different instruments. Direct numeric
   comparison of correlation coefficients across datasets may be misleading.
2. **Correlation ≠ causation.** @eq-pearson measures linear association only.
   No causal inference is warranted from these results.
3. **Sample sizes vary greatly.** The remote work dataset has only 100
   observations, making its PCA results (see @sec-remote) less reliable than
   those for the productivity dataset (n = 20 000).
4. **Missing variables.** Stress, sleep, and activity are consistently
   important, but many potential confounders (age structure, socioeconomic
   status, cultural context) are absent from most datasets.

# Conclusion

This analysis examined five datasets related to human well-being, stress, and
productivity using Pearson correlation (@eq-pearson) and PCA
(@eq-evr). The standardisation step (@eq-zscore) ensured that PCA results
are scale-invariant.

Across all five datasets (summarised in @tbl-summary and @tbl-datasets), a
consistent picture emerges: **sleep quality and duration**, **stress level**,
**physical activity**, and **work-life balance** are the variables most
frequently and most strongly associated with the respective target variables.
These factors appear in contexts as different as student life, clinical sleep
studies, and remote work, which makes the pattern more robust.
