Analysis of Factors Affecting Well-Being

Author

Vladyslav Rudenko

1 Introduction

The aim of this project is to examine selected factors that may be related to well-being, stress, and productivity. The analysis is based on several datasets that describe different groups of people and different situations, such as student lifestyle, sleep and health, productivity habits, and remote work.

The datasets were not combined into one table, because they do not represent the same population and they do not contain the same variables. Instead, each dataset was analyzed separately. After that, the results were compared in order to identify factors that appear repeatedly across different contexts.

The main focus of the project is on lifestyle and personal condition variables, especially sleep, stress, physical activity, daily habits, productivity, and work-life balance. These factors were chosen because they are commonly connected with mental well-being and performance in everyday life.

Two statistical methods were used in the analysis:

  • correlation analysis,
  • Principal Component Analysis.

Correlation analysis was used to study direct relationships between the selected target variable and the other numeric variables. Principal Component Analysis was used to study the internal structure of the datasets and to identify the variables that contribute most strongly to the main patterns in the data.

The results should be interpreted as statistical associations. They do not prove that one variable directly causes another one.

2 Datasets

The project works with cleaned versions of the original datasets. Each dataset has its own target variable.

Dataset file Dataset description Target variable
clean_pca.csv General lifestyle and wellness data wellness
clean_productivity.csv Productivity and daily habits productivity
clean_sleep.csv Sleep and health data stress_level

The separation of datasets is important. For example, student lifestyle data and remote work data may both contain variables connected with stress or productivity, but the meaning of these variables is not exactly the same. Analyzing them separately gives a cleaner and more reliable interpretation.

3 Methods

3.1 Correlation analysis

For every dataset, correlations were calculated between numeric variables and the selected target variable.

A positive correlation means that higher values of one variable are usually connected with higher values of the target variable. A negative correlation means that higher values of one variable are usually connected with lower values of the target variable.

For example, if sleep duration has a positive correlation with wellness, then higher sleep duration is associated with higher wellness values in the dataset. If stress has a negative correlation with wellness, then higher stress is associated with lower wellness values.

Correlation analysis is useful as a first step because it gives a direct overview of the strongest relationships with the target variable. However, it only measures pairwise relationships and does not describe the full structure of the dataset.

3.2 Principal Component Analysis

Principal Component Analysis is a method used to reduce the dimensionality of data. It transforms the original variables into new variables called principal components.

The first principal component, PC1, captures the largest possible amount of variation in the dataset. The second principal component, PC2, captures the largest remaining part of variation, while being independent from PC1. The same idea continues for the following components.

In this project, PCA was used mainly for three purposes:

  • to check how much variance is explained by the first components,
  • to identify which variables contribute most strongly to PC1,
  • to visualize the data in a two-dimensional PCA space using PC1 and PC2.

The explained variance plot shows how much information is contained in each principal component. The PC1 loadings plot shows which original variables are the most important for the main direction of variation. The PC1 vs PC2 plot gives a simplified visual representation of the dataset.

4 Dataset 1: Lifestyle and wellness

Target variable: wellness

4.1 Correlation with wellness

The correlation plot shows how individual lifestyle variables are associated with the wellness score.

Variables with positive correlation values are connected with higher wellness values. Variables with negative correlation values are connected with lower wellness values. The strongest positive and negative correlations are the most important for the first interpretation of the dataset.

This part of the analysis helps identify which lifestyle factors have the clearest direct relationship with well-being.

4.2 PCA explained variance

The explained variance plot shows how much variation is captured by each principal component.

If the first few components explain a large part of the total variance, then the dataset has a structure that can be described using fewer dimensions. This is useful because the original dataset may contain several variables, but the main information can often be summarized by a smaller number of components.

4.3 PC1 loadings

The PC1 loadings plot shows which variables have the strongest influence on the first principal component.

Variables with larger absolute loading values contribute more strongly to PC1. The sign of the loading indicates the direction of the contribution. Variables with the same sign move in a similar direction within the main pattern, while variables with opposite signs move in opposite directions.

This plot is important because PC1 represents the dominant pattern in the lifestyle dataset.

4.4 PC1 vs PC2

The PC1 vs PC2 plot shows the observations after projection into a two-dimensional PCA space.

Each point represents one observation from the dataset. The horizontal and vertical axes are not original variables, but new PCA components. This plot gives a simplified view of the dataset and may show whether the observations form visible groups, trends, or separated areas.

5 Dataset 2: Productivity

Target variable: productivity

5.1 Correlation with productivity

This plot shows the relationship between productivity and the other numeric variables in the productivity dataset.

Positive correlations indicate variables that tend to increase together with productivity. Negative correlations indicate variables that tend to decrease when productivity increases.

The goal of this part is to identify which daily habits or personal factors are most closely connected with productivity.

5.2 PCA explained variance

The PCA explained variance plot shows how much of the dataset structure is captured by each principal component.

If the first components explain a noticeable part of the total variance, then productivity-related data can be interpreted through a smaller number of main patterns instead of many separate variables.

5.3 PC1 loadings

The PC1 loadings plot identifies the variables that define the strongest pattern in the productivity dataset.

If several lifestyle-related variables have strong loadings, this suggests that productivity is not connected with only one isolated factor. It may instead be related to a broader combination of habits and personal conditions.

5.4 PC1 vs PC2

The PC1 vs PC2 plot gives a two-dimensional representation of the productivity dataset.

This visualization is useful for checking whether the dataset has a clear structure after dimensionality reduction. It can also help detect possible groups of observations or unusual points.

9 General comparison

After analyzing all datasets separately, several types of variables appear repeatedly as important:

  • stress-related variables,
  • sleep-related variables,
  • physical activity,
  • daily habits,
  • productivity-related factors,
  • lifestyle or work-life balance.

These factors appear in different datasets and in different contexts. This makes the result more meaningful, because the same general types of variables are connected with well-being, stress, or productivity in more than one source of data.

The comparison also shows why it was useful to analyze the datasets separately. Each dataset has its own target and its own context, but some broader patterns are still repeated.

10 Limitations

There are several limitations in this project.

First, the datasets come from different sources. This means that variables may be measured in different ways, and the results cannot be compared perfectly.

Second, the analysis is based mainly on correlation and PCA. These methods are useful for finding relationships and patterns, but they do not prove causality.

Third, some variables may be influenced by external factors that are not included in the datasets. For example, stress or productivity can depend on personal, social, health-related, or environmental conditions that are not always measured.

Because of these limitations, the results should be interpreted as an exploratory analysis rather than a final explanation of well-being or productivity.

11 Conclusion

The analysis shows that well-being, stress, and productivity are connected with several recurring groups of variables.

The most important repeated factors are sleep, stress, physical activity, daily habits, and lifestyle or work-life balance. These factors appeared in more than one dataset and were visible either through correlation analysis or PCA.

Correlation analysis helped identify variables that are directly associated with the target variables. PCA provided a broader view of the structure of each dataset and helped identify the variables that form the main patterns.

Overall, the results suggest that human well-being and productivity are not connected with only one variable. They are more likely related to a combination of lifestyle, mental-state, and work-related factors.
